## 启动方式

1. 确认 Docker Desktop 已启动，Dify 已停止。
2. 确认项目根目录 `.env` 已填写 `TAVILY_API_KEY`，不要公开该文件。
3. 在 PowerShell 中执行：

```powershell
cd C:\Users\86139\code\resource-agent-demo
docker compose up -d --build
```

4. 打开 `http://localhost:3000`。
5. 查看状态：

```powershell
docker compose ps -a
```

6. 停止 Demo（保留数据库和模型卷）：

```powershell
docker compose down
```
